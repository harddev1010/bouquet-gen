# Poster logo

Place **MAMALOVES_Logo_Liggend.png** here for the MamaLoves logo to appear on generated poster PDFs.

The logo is centered horizontally (9.1 cm inset from each side of the page), sits 2.1 cm above the bottom edge, and is tinted to **#b4b4b4** when the file is PNG (JPEG is drawn unchanged).

Supported formats: PNG, JPG, PDF.
